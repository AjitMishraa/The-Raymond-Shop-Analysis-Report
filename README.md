# The Raymond Shop Analysis Report

An interactive **Power BI Dashboard** developed to analyze and monitor sales performance for **The Raymond Shop**.  
The dashboard provides meaningful business insights through dynamic KPIs and visual analytics.

---

## 📊 Dashboard Overview

### 1️⃣ Raymond Shop Daily Sales Analytics
This dashboard focuses on daily sales performance and includes:

- MTD Sales Growth
- Day-wise Sales Growth
- Product-wise Growth & Degrowth
- Customer Count Analysis
- State-wise Growth & Degrowth

---

### 2️⃣ SKU Wise Sales
This dashboard provides SKU-level sales analysis with:

- Trending SKU Performance
- SKU-wise Sales Insights

---

### 3️⃣ Raymond Shop YTD Sales Analytics
This dashboard highlights yearly sales performance using:

- YTD Sales Growth
- Brand-wise Growth & Degrowth
- Raymond Shop Performance Overview
- State-wise Growth & Degrowth

---
